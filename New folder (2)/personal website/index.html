HOTEL BILL MANAGEMENT SYSTEM

This C program is a simple billing system for a beverage shop called "One More Sip." It allows the user to select items from a menu, specify quantities, calculates the total price, applies discounts and GST, and generates a detailed bill. The program also saves the transaction details, including customer name and mobile number, to a text file named "mybilling.txt".
